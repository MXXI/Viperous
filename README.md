Viperous
========
Habbo Content Management System (CMS)

What is Viperous?
========
Viperous is an content management system for Habbo from scratch, written in PHP, HTML, CSS & JS.



Requirements:
========
- MariaDB (Win or Linux)
- a PHP Webserver
- An SQL from e.g Phoenix or BcStorm
- A brain :P
